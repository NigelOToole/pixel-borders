# Pixel Borders
### Add pixelated borders to HTML elements and customize with different sizes, colours and styles.

### [Demo and documentation](http://nigelotoole.github.io/pixel-borders/)

---
## Quick start
```javascript
$ npm install pixel-borders --save-dev
```

Import the CSS into your project, add the elements to your HTML and initialize the plugin. 

[Full documentation](http://nigelotoole.github.io/pixel-borders/)
